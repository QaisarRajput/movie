# Contributing

We appreciate your interest in helping improve the movie catalog.

## Why contribute?
This project is a free, ad-free catalog for local media discovery. Contributors help:

- improve the UI
- expand search and language support
- enhance GitHub Pages deployment
- clean up API integration and config

## How to contribute
1. Fork the repository
2. Create a new branch for your changes
3. Open a pull request with a clear description
4. Keep updates focused and easy to review

## What we are looking for
- frontend improvements
- performance and loading fixes
- better local network integration
- improved documentation
- tests and reusable components

## Contributor notice
Looking for contributors! If you want to help, please jump in.

This project is open to anyone who wants to improve the catalog, fix issues, or add features. Your help is welcome.
